Multiplayer Social Deduction Game

An interactive and real-time Multiplayer Social Deduction Game built using TypeScript, JavaScript, and Tailwind CSS. Players can join rooms, communicate, complete objectives, and identify hidden imposters in a fast-paced multiplayer environment.

🚀 Features

🎮 Real-time multiplayer gameplay.
👥 Create and join private game rooms.
💬 Live chat and player communication.
🕵️ Social deduction mechanics.
⚡ Real-time game state synchronization.
🧠 Role-based gameplay system,
🗳️ Voting and elimination system.
📱 Fully responsive UI,
🎨 Modern interface with Tailwind CSS.
🔐 Secure authentication and room management.

🛠️ Tech Stack

Frontend: TypeScript, JavaScript
Styling: Tailwind CSS, CSS3
Realtime Communication: Socket.io / WebSockets
Backend: Node.js, Express.js
Database: MongoDB 

🎮 Gameplay Overview

Players join a multiplayer lobby .
Each player receives a secret role.
Crewmates complete objectives while identifying imposters.
Imposters sabotage and eliminate players secretly.
Players discuss and vote to eliminate suspicious players.
The game ends when objectives are completed or imposters win.
